Quest Log Application

HTML, Advanced CSS, Django and Python

Disclaimer

This project is inspired by Nintendo's Legend of Zelda series and is for personal use only. The application is not released to the public and is not intended for commercial use. All original logos, icons, and sounds are property of Nintendo.

Background

As a video game lover, I am fond of the organization, motivation, and accomplishment of completing tasks, quests, stories, and objectives that various games have to offer. When I started college, I sought to find an application to help manage my homework schedule that worked like a to-do list but felt like an adventure log or quest menu. Unfortunately, none of the pre-existing apps I tried really stuck with me, which is why I decided to code my own.

Personally, my favorite adventure menu is the one from Nintendo’s Legend of Zelda: Breath of the Wild due to its modern style, the confirmation noises that are generated when a task is created/completed, and the ability to “set destination” to highlight the current task. These were all things that I looked for in a game-themed planner, which is why I decided to create a functional version based on this menu. While the main purpose behind this project is to have an application personal to me that will help keep me motivated in my work, I’m hoping that other video game lovers will appreciate and have a desire to use this application as well.

I chose to utilize Django and python to create the application. Quests are stored under a database and are only brought into view based on the user that is currently logged in. Django's views and urls functionality allowed for organized creation and display of quests, quest details, and CRUD functionalities within the database. View the video at the bottom of the page to see how the project came together!

The Project

The features of this application in it's current state include:

  • The ability to add, edit, delete, and mark quests as complete.
  • When a quest is marked as "active" it will be displayed at the top of the list.
  • Sounds will accompany actions, such as selecting a quest or displaying a celebratory "complete" banner when a task is completed.
  • Completed tasks are tallied as a quest counter represented by "Rupees", the in-game currency.
  • User creation for log in/out functionality, which only displays tasks pertaining to the active user.
  • Mobile and desktop view.
